Part 1: Return to Programming

So, a year had passed since I worked on AGBE (or really anything programming related in general), and I had finally decided to come back to it. This time, I had a lot more knowledge about the Gameboy than I did when I started. So much in fact, that AGBE Rev 1's code was so broken, I decided to start over entirely. This resulted in AGBE Rev 2, also just known as AGBE.
